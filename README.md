## 一、先搞清楚：biggerbox 到底是什么

biggerbox 的全称是 **BIGGERBOX-20-1024**，是搬瓦工 2024 年底推出的 Box 基础系列限量版套餐之一，属于 DC99 机房。当时 Box 系列最初需要邀请码才能购买，后来逐步开放。

它的完整配置如下：

- CPU：1核 Intel Xeon
- 内存：1024 MB（1 GB）
- 硬盘：20 GB SSD（RAID-10）
- 月流量：1000 GB（1 TB）
- 带宽：1 Gbps
- 机房：美国洛杉矶 DC99
- 线路：三网回程 CN2 GIA
- 价格：**$37/年**
- 不支持机房迁移

和它的兄弟 MiniBox（$29/年，512MB 内存/10GB 硬盘/500GB 流量）相比，biggerbox 在内存、硬盘、流量三项上全部翻倍，多花 8 美元，值不值看你一眼就知道了。

### 用户类型一：预算有限、初次体验，首选入门

**代表套餐：MiniBox / BiggerBox（DC99 基础系列）**

DC99 是搬瓦工专门用来测试新线路的洛杉矶机房，三网回程接入 CN2 GIA，延迟表现不错，但机房目前不支持迁移，稳定性方面也只提供"尽力保障"（best effort），没有 SLA 承诺（后来陆续升级为 99.9% 在线率保证）。

对于第一次用搬瓦工、想低成本试水的朋友，这个系列性价比最高——

- **MiniBox**（$29/年）：512MB 内存，适合轻度使用，比如搭个博客或跑个脚本
- **BiggerBox**（$37/年）：1GB 内存，流量翻倍，适合日常建站或轻量代理

值得一提的是，MiniBox 和 BiggerBox 之间支持在后台自助升级，BiggerBox 也可以升级到 PowerBox，差价补一下就行，不用重新购买，非常方便。

### 用户类型二：有一定预算、追求稳定线路，中端进阶用户

**代表套餐：BiggerBox Pro / MegaBox Pro（DC1 Box Pro 系列）**

如果你觉得 DC99 的"尽力保障"不够放心，想要更稳、更快的线路，那 Box Pro 系列就是为你准备的。

DC1 是搬瓦工另一个洛杉矶机房，硬件规格全面升级：CPU 换成了 AMD EPYC，带宽直接翻倍到 2.5 Gbps，存储仍是 RAID-10 SSD。更重要的是线路：

- 电信回程：CN2 GIA 直连

**biggerbox 和 biggerbox pro 到底差在哪？**

两者配置数字看起来差不多（都是 1GB 内存 / 20GB 硬盘 / 1TB 流量），但核心差异是：

- 机房不同：DC99 vs DC1（DC1 更新更好的硬件，AMD CPU，RAID-10）
- 线路不同：DC99 是三网 CN2 GIA，DC1 是 CN2 GIA + CMIN2（移动更快）
- 带宽不同：1 Gbps vs 2.5 Gbps
- 价格差：$37 vs $39，差 2 美元

几乎所有指标 DC1 都更强，价格只多 2 美元，Pro 版性价比明显更高。

**Box 系列可以迁移机房吗？**

目前 Box 系列全系不支持机房迁移，购买后只能使用指定机房。如果有多机房切换需求，建议考虑搬瓦工的 THE PLAN 系列或 CN2 GIA-E 套餐。
